Detailed Description

Describes a type.

In .NET/CLI, a type is completely described by an instance of System.Reflection.Type. However, for System# this information is not sufficient. E.g. we consider a Signed value of 12 bits length to have a different type than a Signed value of 16 bits length. This is solved by so-called type parameters. A type parameter is a property of a type whose per-instance value is considered to contribute to its type. Thus, type descriptors provide a more detailed type system which was specifically designed for hardware/embedded modelling.
