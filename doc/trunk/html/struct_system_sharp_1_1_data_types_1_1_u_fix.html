Converts value to the closest fixed-point number, using intWidth integer bits and fracWidth fractional bits.

If intWidth is chosen smaller than the proper representation of value as a fixed-point number actually requires, an arithmetic overflow will occur. The behavior of this method in the presence of overflows depends on the currently selected overflow mode (

See Also
FixedPointSettings

Definition at line 385 of file UFix.cs.

static UFix SystemSharp.DataTypes.UFix.FromSLV ( StdLogicVector  slv,
int  fracWidth 
)
static

Interprets logic vector slv as a fixed-point number with fracWidth fractional bits. The total width matches the length of the logic vector.

Logic values of '1' and 'H' are interpreted as ones, all other values are interpreted as zeroes. Please keep this in mind, since logic vectors containing logic values such as 'Z' or 'X' may lead to unexpected results.
