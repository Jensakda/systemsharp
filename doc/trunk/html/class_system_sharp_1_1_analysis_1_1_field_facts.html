bool SystemSharp.Analysis.FieldFacts.IsSubMutated
getset

Whether any subordinate field of this field is overwritten during runtime.

Consider a struct: struct InnerStruct { public int X; }

class A { public InnerStruct S;

public static void Main() { var a = new A(); a.S.X = 5; } } For field A.S, we would get: IsWritten = false, IsSubMutated = true. For field InnerStruct.X, we would get: IsWritten = true, IsSubMutated = false.
