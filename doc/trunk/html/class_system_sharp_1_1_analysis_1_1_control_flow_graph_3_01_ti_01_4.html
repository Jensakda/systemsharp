Creates a basic block for a range of instructions. Override this if you want to create a specialized basic block implementation.

This method does not check whether the given instruction range formally meets the criterion of a basic block, e.g. having no conditional branches except for the last instruction. It is up to the caller to supply valid arguments.

virtual bool SystemSharp.Analysis.ControlFlowGraph< Ti >.IsLocalPinned ( int  local)
virtual

Tells whether a local variable is pinned, see http://msdn.microsoft.com/en-us/library/f58wzh21%28v=vs.110%29.aspx

Default implementation returns always false. Override method to implement desired behavior. This feature was never tested in reality and probably should be removed in future releases, since pinned variables are only needed in unsafe code, and unsafe code is a very bad idea for hardware modeling anyway.

Parameters
localindex of local variable
Returns
whether variable is pinned
