T  v2 
)

Resolution function delegate

Depending on your use case, there are multiple possibilities of implementing a resolution function. The most simple and stupid option is to either keep the existing element (return v1 ) or to replace the existing element by the new one (return v2 ). A more advanced approach is to specify some collection or set type, e.g. a HashSet, and to construct the set-union of both parameters.

v2another element
Returns
