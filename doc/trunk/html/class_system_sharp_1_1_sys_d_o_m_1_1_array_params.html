bool SystemSharp.SysDOM.ArrayParams.IsStatic
getset

Whether the constructed array is static.

A static array is an array whose elements are determined during code analysis, at least symbolically. It is possible to revoke the static property of this instance by setting this property to false, but it is not possible to convert a non-static array to a static one.
