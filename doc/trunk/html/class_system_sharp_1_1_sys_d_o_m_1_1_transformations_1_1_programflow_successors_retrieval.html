static IEnumerable<Statement> SystemSharp.SysDOM.Transformations.ProgramflowSuccessorsRetrieval.GetProgramflowSucessors ( Statement  stmt)
static

Returns the set of direct successors of a statement.

This operation is only valid on atomic statements, that is all statement types except CompoundStatement and LoopBlock. Furthermore, only atomic statements constitute the result enumeration.
