Detailed Description

This class represents a signal which transports a resolved data type.

A resolved data type defines a resolution rule which is applied when multiple drives write to the same signal. An example is four-valued logic which allows for modeling of tri-state logic. If one process "drives" the signal to 'Z' (high impedance) and another process drives it to '1', the resulting signal value gets resolved to '1'.
