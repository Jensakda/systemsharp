Detailed Description

Generic interface of control path construction algorithms

Control path construction is the instantiation of an appropriate datapath controller for a fully constructed datapath. The purpose of the controller is to reconfigure the datapath in each c-step as prescribed by the flow matrix. More specifically, it directs the flows between each data source and sink. A good example architecture is the explicit finite state machine implementation.
