IEnumerable<IXILMapping> SystemSharp.Components.FU.IXILMapper.TryMap ( ITransactionSite  taSite,
XILInstr  instr,
TypeDescriptor[]  operandTypes,
TypeDescriptor[]  resultTypes 
)

Tries to map XIL instruction instr to transaction site taSite .

Typically, the returned sequence contains either one mapping or is empty (because instruction cannot be mapped to given transaction site). However, some commutative operations like addition and multiplication in fact allow for two mapping possibilities by swapping the operands. In those cases, both mappings should be returned.

Parameters
taSitetransaction site to carry out the mapping
instrXIL instruction to map
operandTypesoperand types of instruction
resultTypesresult types of instruction
Returns
a sequence of possible mappings, empty sequence if instruction cannot be mapped
