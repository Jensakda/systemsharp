override bool SystemSharp.Components.Time.Equals ( object  obj)

Tests this object for equality with another object.

Two time representations are defined to be equal iff both unit and value match. Therefore, 0.1sec and 100ms are NOT the same. To check whether two Time instances actually represent the same physical time, use CompareTo() instead.

Parameters
objThe object to which this object should be compared
Returns
true if both objects are equal, false if not
