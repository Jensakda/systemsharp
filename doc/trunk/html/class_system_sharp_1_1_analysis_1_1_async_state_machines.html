static MethodBase SystemSharp.Analysis.AsyncStateMachines.UnwrapEntryPoint ( this MethodBase  method)
static

Retrieves the actual entry point of an async method.

Async methods are compiled as wrapper methods, redirecting the actual work to the MoveNext method of an underlying state machine implementation. That MoveNext will be found and returned.

Parameters
methoda possibly async method
Returns
its actual implementation, i.e. the MoveNext method of its underlying state machine data structure in case of an async method, the method itself in case of a "normal" method.
